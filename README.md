# snake
Snake Game

### DEV:
1. Run `yarn install` to install all dependencies.
1. Run `yarn start` command in the root directory.
2. Run `yarn test` to run unit tests.

### PRODUCTION
1. Run `webpack` command to create `dist/` directory containing the app
